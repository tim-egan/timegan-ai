1
CSUB Primary vs Alternate breaker mismatch — confirmed site-wide (8 of 8 UPS systems, both data halls). On every CSUB-1 (Primary), the first output breaker is 1600AF/1600AT. On every mirrored CSUB-2 (Alternate), it's downsized to 1000AT/1000AF with a 350AR rating plug — confirmed identical across all 4 systems in Data Hall K / Room 5500 (E-6257–60) and all 4 systems in Data Hall J / Room 5100 (E-6251–54). Sheet E-6105 Note 3 explains the Alternate path is a generator-backed maintenance/failure mode (not continuous full UPS load), which supports this being an intentional design standard rather than an error. Still recommend a confirming RFI on the specific load basis and downstream STS/PDU coordination — status downgraded from "likely error" to "design-intent-plausible, pending load calc confirmation."
2
Rack density inconsistency between hall halves (Data Hall K / Room 5500) — now cross-confirmed against architectural drawings. Systems 1&2 (35-rack rows, E-6261) and Systems 3&4 (30-rack rows, E-6262) both report 272.75kW/row — implies ~30% higher kW/rack assumption in one half with no differentiating note. The architectural containment axo (A-4010) and overall plan (A-1010) confirm the physical pairing: 35-rack rows pair into 70-rack/545.5kW HAC pods, while 30-rack rows pair into 60-rack/545.5kW HAC pods — same total kW per containment pod despite the 10-rack difference, which is the root of the discrepancy. Recommend confirming intended per-rack power density before PDU/busway procurement, and checking Data Hall J's (Room 5100) equivalent PDU sheets for the same pattern.
3
STS tag reuse across sheets. STS tags appear on both UPS system sheets and downstream PDU/CTP sheets. Confirmed as the same physical equipment referenced from two views — not duplicates. Flagged only to ensure equipment register de-duplicates correctly across all 8 UPS systems.
✓
HRG grounding on TS2 — RESOLVED, no issue. Confirmed visually on E-6257: TS2 has explicit HRG resistor to ground identical to TS1. Both transformers per system are HRG, consistent across the set. Flag closed.
